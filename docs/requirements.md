# 需求分析文档

## 1. 项目背景

当前很多用户有下载保存视频到本地的需求，但不同视频平台存在以下问题：
- 部分平台不支持下载功能
- 支持下载的平台可能存在限制（清晰度有限、无法批量下载等）
- 手机端下载操作不够便捷
- 跨平台下载需要切换不同的工具

## 2. 目标用户

| 用户类型 | 核心需求 |
|----------|----------|
| 普通用户 | 快速下载各平台视频到本地保存 |
| 内容创作者 | 收集素材、参考视频 |
| 学习者 | 保存教程视频离线观看 |

## 3. 核心需求

### V1.0

| 优先级 | 需求 | 说明 |
|--------|------|------|
| P0 | 多平台视频链接解析 | 粘贴链接即可获取视频信息（标题、时长、封面、格式列表） |
| P0 | 多清晰度选择 | 支持 360p/720p/1080p/4K 等清晰度选择 |
| P0 | 视频下载 | 选择格式后一键下载 |
| P0 | 实时下载进度 | WebSocket 推送下载进度（百分比、速度、ETA） |
| P0 | 文件保存 | 下载完成后提供文件下载链接 |

### V2.0（当前版本）

| 优先级 | 需求 | 说明 |
|--------|------|------|
| P0 | 视频 AI 总结 | 输入视频 URL，AI 生成内容摘要、章节大纲、思维导图 |
| P0 | 字幕/转录展示 | 提取并展示带时间戳的字幕文本（优先 B 站 CC 字幕 API，降级 yt-dlp） |
| P1 | 思维导图 | 将总结内容可视化为交互式思维导图（markmap） |
| P1 | AI 问答 | 基于字幕内容的多轮流式对话 |
| P1 | Markdown 渲染 | AI 输出内容结构化展示（摘要、聊天消息均支持） |

### 未来版本（待规划）

| 优先级 | 需求 | 说明 |
|--------|------|------|
| P2 | 批量下载 | 批量解析和下载多个视频链接 |
| P2 | 付费功能 | 会员订阅、按次付费下载 |
| P2 | 手机适配 | 移动端响应式优化 |

## 4. 非功能性需求

- **易用性**：粘贴链接即可解析，最少操作步骤
- **性能**：解析时间 < 3s，下载速度由平台带宽决定
- **可靠性**：支持 1800+ 视频平台，失败时给出明确错误提示
- **兼容性**：Windows/macOS/Linux 均可运行，主流浏览器支持
- **轻量化**：无数据库依赖，内存存储任务状态

## 5. 约束条件

- 使用 yt-dlp 作为核心视频引擎（不做自研下载器）
- 仅供学习使用，尊重版权
- 前端采用 **SaveAny 品牌风格**：深蓝黑渐变背景（#0F172A）、蓝青渐变主色（#3B82F6 → #06B6D4）、毛玻璃卡片、简洁现代的暗色主题
- 目标用户以国内为主，平台展示和文案国内优先
- 需具备吸引人付费的文案和视觉设计（Pro 标识使用粉色 #EC4899）
